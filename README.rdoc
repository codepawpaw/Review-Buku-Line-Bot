== Review Buku Line Bot

A Virtual Asisstant in Line Messenger for searching book rating.
This repo shows you a practical use of the Line Messaging API.

This project use ruby on rails without active record

== Getting Started

`````
$ clone this repo
$ bundle install
$ rails s
`````
